Markdown 是一种轻量级标记语言，让你可以使用易读易写的纯文本格式编写文档。这篇指南将展示 Markdown 的各种语法。

## 标题

Markdown 支持 6 级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本样式

你可以使用不同的文本样式：

- **粗体文本**：用双星号或双下划线包裹
- *斜体文本*：用单星号或单下划线包裹
- ***粗斜体文本***：用三星号包裹
- ~~删除线文本~~：用双波浪线包裹
- `行内代码`：用反引号包裹

## 列表

### 无序列表

使用星号、加号或减号创建无序列表：

* 第一项
* 第二项
* 第三项

### 有序列表

使用数字加句点创建有序列表：

1. 第一项
2. 第二项
3. 第三项

### 嵌套列表

列表可以嵌套：

1. 第一项
    * 第一项的子项
    * 第一项的子项
2. 第二项
    * 第二项的子项
        1. 子项的子项
        2. 子项的子项

### 任务列表

可以创建待办事项列表：

- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个待办事项

## 引用

使用大于号创建引用块：

> 这是一段引用文本。
>
> 引用可以包含多个段落。
>
> > 引用还可以嵌套。

引用中可以包含其他 Markdown 元素：

> **粗体**
>
> *列表项*
>
> `代码`

## 代码

### 行内代码

在段落中插入`行内代码`，使用单个反引号。

### 代码块

使用三个反引号创建代码块，可以指定语言：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
def greet(name):
    print(f"Hello, {name}!")

greet("World")
```

### 不含语法高亮的代码块

缩进 4 个空格也可以创建代码块：

    这是普通文本
        这是代码块
```

## 分隔线

使用三个或更多的星号、减号或下划线创建分隔线：

---

***

___

## 表格

使用竖线和连字符创建表格：

| 左对齐 | 居中 | 右对齐 |
| :--- | :---: | ---: |
| 内容1 | 内容2 | 内容3 |
| 长内容 | 中等 | 短 |

### 不规范但可读的表格

| 标题1 | 标题2 | 标题3 |
|---|---|---|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

## 转义字符

如果需要显示 Markdown 特殊字符，可以使用反斜杠转义：

\*不是斜体\*
\[不是链接\]
\`不是代码\`

需要转义的字符：
- \ 反斜杠
- ` 反引号
- * 星号
- _ 下划线
- {} 花括号
- [] 方括号
- () 圆括号
- # 井号
- + 加号
- - 减号
- . 句点
- ! 感叹号

## HTML 支持

Markdown 支持嵌入 HTML：

<details>
<summary>点击展开隐藏内容</summary>

这是隐藏的详细内容。
</details>

### 自定义容器

Hugo 支持自定义的 shortcode 容器，可以创建各种提示框：

{{< admonition type="note" title="注意" >}}
这是一个提示框。
{{< /admonition >}}

## 脚注

可以使用脚注：

这里有一个脚注引用[^1]。

[^1]: 这是脚注的内容。

$$
\frac{n!}{k!(n-k)!} = \binom{n}{k}

## 最佳实践

1. **保持简洁** - Markdown 的优势在于简洁易读
2. **适当的空行** - 在不同元素之间添加空行
3. **一致的格式** - 在整个文档中保持一致的格式
4. **测试渲染** - 经常预览确保效果符合预期
5. **使用工具** - 使用专门的 Markdown 编辑器提高效率
